vscode设置console.log快捷键 #

  • 第一步：文件 - 首选项 - 用户代码片段 ；

  • 第二步：选择 JavaScript；

  • 第三步：

{
	"Print to console": {
		"prefix": "cl",
		"body": [
			"console.log('$1');",
			"$2"
		],
		"description": "Log output to console"
	}
}

vscode插件-prettier-code formatter #

  • 在根目录增加prettier配置文件 .prettierrc
  • 将下面配置copy进去保存
  "bracketSpacing": true,
  "printWidth": 160,
  "tabWidth": 2,
  "useTabs": true, //使用tab换行
  "semi": true, //有无分号
  "singleQuote": true,
  "trailingComma": "none"
} ``` 

- 右键-格式化文档，会按照这个配置进行格式化
